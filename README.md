# httpbin(1): HTTP Request & Response Service


Support **multi-arch**(arm64/aarch64, amd64/x86_64).

Run locally:
```sh
docker pull huaxlin/httpbin
docker run -p 80:80 huaxlin/httpbin
```

Docker Image: https://hub.docker.com/r/huaxlin/httpbin/
